1. Resolución recomendada: 1280×800 mínimo.
2. Recortar para mostrar únicamente la acción relevante.
3. Sustituir todos los datos personales reales por demostrativos:
   - Nombre: Juan Pérez / Ana Gómez
   - CI: 1234567890
   - Placa: ABC-1234
   - Correo: ejemplo@demo.com
4. No mostrar contraseñas, API Keys ni URLs de Supabase.
5. No capturar la consola del navegador.
6. Guardar en formato PNG con nombre exacto indicado.
